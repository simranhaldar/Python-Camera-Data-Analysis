# 📷 Camera Data Analysis Using Python

## Project Overview

This project analyzes a dataset of approximately 1000 camera models and performs data cleaning, preprocessing, and exploratory data analysis using Python.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter/Python IDE

---

## Dataset Information

The dataset contains:

- Camera Model
- Price
- Weight
- Resolution
- Zoom Range
- Storage Included
- Release Year
- Dimensions

---

## Tasks Performed

### Data Cleaning

- Checked missing values
- Replaced blanks with NaN
- Filled missing values using median

### Feature Engineering

- Added Discounted_Price column
- Applied 5% discount on original price

### Data Manipulation

- Dropped unnecessary columns
- Renamed columns
- Updated incorrect model names

### Data Analysis

- Most expensive camera
- Least weight camera
- Cameras released in 2005 and 2006
- Cheapest and most expensive camera in 2007
- Year with maximum camera releases
